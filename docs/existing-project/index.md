# 已有项目工作流

> 30 秒了解：在已有代码库中安装 yy-spec，AI 会先理解现有架构再开始工作。

## 与新项目的区别

已有项目的关键区别是：AI 需要先理解你现有的代码结构、技术栈和架构约定，然后在此基础上工作。

| 步骤 | 新项目 | 已有项目 |
|------|--------|---------|
| 初始化 | 空项目建立上下文 | 分析现有代码建立上下文 |
| 新功能 | 从零设计 | 基于现有架构扩展 |
| Bug 修复 | 相同 | 相同 |
| 调查问题 | 相同 | 可利用现有代码分析 |
