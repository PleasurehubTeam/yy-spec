# Product Overview

yy-spec 文档站 — 一个基于 VitePress 的通用文档项目，服务于 AI 开发者和团队。

## Core Capabilities

- **工具使用文档** — 以应用场景为分类，提供 yy-spec 的使用指南和命令参考
- **AI 开发知识库** — 涵盖 AI 驱动开发生命周期（AI-DLC）、规格驱动开发等方法论
- **多语言支持** — 中文为主，兼顾英文内容

## Target Use Cases

- yy-spec 新用户快速上手（由浅入深，面向零基础读者）
- 团队成员查阅工作流命令和自定义配置
- AI 开发实践者学习规格驱动开发方法论

## Value Proposition

不仅仅是工具文档，而是一个 AI 开发知识的综合平台。以场景驱动的方式组织内容，降低学习门槛。
